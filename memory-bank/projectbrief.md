# Project Brief

## Core Goals
- Create an interactive portfolio showcasing AI-powered games and tools.
- Port original Python-based AI implementations (Tic-Tac-Toe, Minesweeper, Six Degrees) to JavaScript/React.
- Demonstrate expertise in AI algorithms (Minimax, Logical Deduction, BFS).
- Provide a platform for generating AI-powered renovation quotes (Presupuestos Reformas).

## Scope
- **AI Games**: Tic-Tac-Toe, Minesweeper, Six Degrees of Kevin Bacon.
- **Demos**: Lead Qualifier, Presupuestos Reformas.
- **Frontend**: Responsive React/Vite application.
- **Backend**: Serverless functions (Vercel) and Express server for local development/persistence.
- **Multi-language**: Support for English, Spanish, and Catalan.
