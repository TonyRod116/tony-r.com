# Product Context

## Why It Exists
The project serves as a professional portfolio for Tony Rodríguez, a Software Engineer with AI expertise. It aims to bridge the gap between complex AI algorithms and intuitive, interactive web applications.

## Problem It Solves
- **Abstract AI Concepts**: Makes algorithms like Minimax and BFS tangible through playable games.
- **Lead Generation & Quoting**: Provides a practical demo for businesses (specifically renovation companies) to automate lead qualification and initial quoting via AI.
- **Portability**: Moves Python AI logic to the web (JS/React) for wider accessibility.

## Target Audience
- **Potential Employers/Clients**: Looking for proof of AI and web engineering skills.
- **AI Enthusiasts**: Interested in seeing real-world implementations of logical deduction and search algorithms.
- **Business Owners**: Seeking automation for sales and quoting processes.
